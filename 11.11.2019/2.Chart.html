Creating a chart

To create a chart, we need to instantiate the Chart class.To do this, we need
to pass in the 2nd context of where we want to draw the chart. Here's an
example.

// Get the context of the canvas element we want to select
var ctx = document. getElementById ("MyChart") .getContext("2d");
var myNewChart = new Chart(ctx).PolarArea(data);

We can also get the context of our canvas with JQuery. To do this,
we need to get the DOM node out of the JQuery collection and call the
get Context("2d") method on that.

// Get the context with JQuery - using JQuery's get() method
varctx = $(#MyChart).get(0).getContext ("2d");
var myNewChart = new Chart(ctx)
